**Description:**

-  applique la fonction : TAN :math:` tan{z} `

\*Méthode d'itération:\*

-  point par point

\*Dépendances:\*

-  un champ météorologique

\*Résultat(s):\*

-  le champ météorologique passé en entrée modifié

\*Algorithme:\*

-  pour chaque point applique l'algorithme choisi

\*Références:\*

-  man 3 tanf

\*Mots clés:\*

-  UTILITAIRE/UTILITY, tangente/tangent

\*Usage:\*

**Exemple d'appel:**
