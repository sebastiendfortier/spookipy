** English


*Description:*

- Applies the absolute value to each element of a field

*Iteration method:*

- Point-by-point

*Dependencies:*

- A meteorological field

*Result(s):*

- The absolute value of the meteorological field

*Algorithm:*

- Applies the function |z| to each value (z) of a given field

*Reference:*

- Does not apply
